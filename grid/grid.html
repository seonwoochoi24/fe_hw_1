flex 외우기 어려움
grid 나누기 숫자 계산이 헷갈림
  • 기본 템플릿은 !+Tab
  • border: 1px solid black; 쓰면 테두리 확인하기 편함
  • link href="../style/style.css" 쓰면 외부 스타일 가능
  • display: flex; flex-direction: 메인 축;
  • justify-content: 메인축 방향 정렬
  • align-items: 교차축 방향 정렬
  • align-content: 수직 방향 정렬 결정
  • flex-wrap: 줄넘김 처리 결정
  • flex-basis: 아이템 기본 크기 설정
  • nth-child(n): n번째 아이템에 속성 입힘
  • lorem: 더미 데이터 만들기
  • grid-template-rows/columns: 행높이/열너비
  • (row/column-)gap: 간격
  • grid-template-rows/columns: grid 트랙의 크기 지정
